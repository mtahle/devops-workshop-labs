CI/CD for ML
with GitHub Actions

Automate test → train → build → push. Every push to main becomes a validated, versioned container.

The problem

What CI/CD eliminates

"It works on my machine" — automated tests catch breakage before it reaches anyone else.
Forgetting to run tests before pushing — the pipeline runs them automatically on every commit.
Manual Docker builds with inconsistent tags — the pipeline builds and tags with the exact commit SHA.
Untested model accuracy drops — pytest checks that accuracy stays above threshold on every run.
Result: bad code never reaches production because the pipeline is the gatekeeper.

GitHub Actions

A YAML file that lives next to your code

Place a .github/workflows/ci.yml in your repo and GitHub runs it automatically on the events you define.

Trigger (on:)

Defines WHEN the pipeline runs: push, pull_request, schedule, or manually via the GitHub UI.

Job

A group of steps that runs on one fresh machine (ubuntu-latest). Jobs can run in parallel or in sequence.

Step

The smallest unit of work. uses: runs a pre-built action. run: executes any shell command.

.github/workflows/ci.yml

The ML CI pipeline

name: ML CI Pipeline
on:
  push:
    branches: [main]

jobs:
  test-and-build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v4
        with: { python-version: '3.11' }
      - run: pip install -r requirements.txt
      - run: python model.py                # train
      - run: pytest tests/ -v              # test accuracy
      - run: docker build -t iris-classifier:${{ github.sha }} .
tests / test_model.py

Testing an ML model with pytest

import pickle, numpy as np, pytest

@pytest.fixture
def model():
    with open("model.pkl", "rb") as f: return pickle.load(f)

def test_model_loads(model):      assert model is not None

def test_predicts_setosa(model):
    X = np.array([[5.1, 3.5, 1.4, 0.2]])
    assert model.predict(X)[0] == 0         # setosa

def test_accuracy_above_90(model):        # THE gate
    from sklearn.datasets import load_iris
    from sklearn.metrics import accuracy_score
    X, y = load_iris(return_X_y=True)
    assert accuracy_score(y, model.predict(X)) >= 0.90
